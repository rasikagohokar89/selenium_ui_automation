# Selenium Automation Basics

This project demonstrates core **Selenium WebDriver automation concepts** implemented using **Python**.  
It serves as a learning and reference project for building scalable UI automation frameworks.

---

## Tech Stack

- Python
- Selenium WebDriver
- ChromeDriver
- Page Object Model (POM)

---

## Covered Selenium Concepts

This repository covers fundamental Selenium automation topics including:

- WebDriver setup
- Locators (ID, Name, XPath, CSS Selector)
- Explicit and Implicit Waits
- Handling Alerts
- Handling Frames & iFrames
- Multiple Windows / Tabs
- Dropdown handling
- Mouse Actions
- viewports
- webtables
- Rest API
